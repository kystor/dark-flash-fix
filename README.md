# 定义内容

## flash-guard更加好用，它是一款轻量且强大的浏览器扩展，旨在彻底解决暗黑模式下网页加载或跳转时出现的瞬间“白屏闪烁”痛点。点击扩展即可打开功能菜单。

## ✨ 核心特性
- **瞬时拦截**：在浏览器解析网页前的第一毫秒注入深色底色，物理级消灭白闪。
- **雷达侦测**：实时监控 DOM 变化，网页内容真正加载出后瞬间撤退。
- **丝滑过渡**：采用“交叉溶解”技术，背景色渐变结合透明度淡出，视觉体验极佳。
- **视力保护**：支持切回标签页时自动触发柔和过渡，让眼睛适应光线变化。

## 🛠️ 安装步骤（新手详细版）

### 1. 下载与解压
1. 下载 `flash-guard` 压缩包。
2. 将其**解压**到电脑上一个你不会轻易移动或删除的文件夹中。
   - *注意：安装后请保留该文件夹，删除文件夹会导致扩展失效。*

### 2. 进入扩展管理页面
打开你的浏览器，在地址栏输入以下对应地址并回车：
- **Chrome 浏览器**: `chrome://extensions/`
- **Edge 浏览器**: `edge://extensions/`

### 3. 开启开发人员模式
在打开的页面中：
- **Chrome**: 找到右上角的 **“开发者模式”** 开关并打开。
- **Edge**: 找到左侧边栏（或左下角）的 **“开发人员模式”** 开关并打开。

### 4. 加载扩展程序
1. 点击页面上方出现的 **“加载解压缩的扩展”**（Edge）或 **“加载已解压的扩展程序”**（Chrome）按钮。
2. 在弹出的文件选择框中，**选中你刚才解压后的整个文件夹**。
3. 点击“选择文件夹”或“确定”。

### 5. 验证安装
- 现在扩展卡片应该已经出现在列表中。
- 你可以访问如 [LeafAAS Login](https://www.leafaas.com/login) 等网页，或者尝试在不同标签页之间切换，感受丝滑的防闪加载效果。

## 💡 开发与调试
如果你修改了 `content.js` 中的参数（如颜色或时间）：
1. 回到浏览器的 `extensions` 页面。
2. 找到 Flash-guard 卡片。
3. 点击右下角的 **“刷新”** 图标即可应用最新代码。
